Ansible-Role-PHP
=========
This is a simple role which installs PHP 7.2/7.3/7.4

Requirements
------------
None

Role Variables
--------------
php_version: "7.4"

Dependencies
------------
None

Example Playbook
----------------
  roles:
    - role: ansible-role-php
      vars:
        php_version: "7.4"
